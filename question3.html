Question 3:

Review 3 different GUI Git clients

Answer:

Git is an extremely well supported product, with a variety of clients for almost every operating system.
There are a number of different forms of git client available:

  • GUI: Graphical user interface based standalone client
  • CLI: Command line interface clients which operatate from the terminal
  • Add-on: Which enhances the functionality of an existing program, normally through a plugin system

The three git GUI clients I will review are:

  1. Git Kraken
  2. Gitg
  3. Smart Git

Git Kraken

A highly efficient, intuitive cross-platform git client.
The ability to visualize git commits, pointer linkages, and with their respective tags is especially useful.

Has numerous features to enhance productivity including:

  • Hints and visual interactions
  • Multiple profile support
  • Incorporated merge tool
  • Intelligent and fast search tool
  • Undo and redo functions with a single click
  • Github and Bitbucket integration

Gitg

A Linux and Windows client originally designed for the GNOME desktop manager. It has an uncluttered GUI with shell integration. Only has support for Git.

Some of the useful features include:

  • Native shell integration
  • Ability to browse history and recently used repositories
  • Highly configurable
  • Fully featured and separate staging area to view and perform commits

SmartGit

Very popular and powerful cross-platform git GUI client for Linux, Windows and Mac OS X.
Aimed at professionals, it has numerous features to boost the users productivity through optimised workflows

Contains the following features as standard:

  • Excellent integration with Github, Bitbucket and Atlassian Stash
  • SVN repository support
  • Git pull requests and comments are supported
  • Git-flow, file comparison and merge tools along with an SSH-client are included
