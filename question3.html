Question 3:

Review 3 different GUI Git clients

Answer:

Git is an extremely well supported product, with a variety of clients for almost every operating system.
There are a number of different forms of git client available:

  • GUI: Graphical user interface based standalone client
  • CLI: Command line interface clients which operatate from the terminal
  • Add-on: Which enhances the functionality of an existing program, normally through a plugin system

The three git GUI clients I will review are:

  1. Git Kraken
  2. Git Cola
  3. TortoiseGit
