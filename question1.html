Question 1:

Argue 2 advantages of Git DVCS over Dropbox


Answer:

Git Advantages:

  1. Versioning system
  2. Concurrent editing by multiple users

1 ‐ The versioning system ....

Overview

When working in software development, with files which change frequently, there are a number of advantages to having a version control system:

  • Enables different versions of the same file to exist
  • Allows the developer to easily roll back changes
  • Removes the need to rename the different versions of the same file

2 ‐ Concurrent editing by multiple users ....

When working in software development, where teams of developers can be working on the same files, there are a number of advantages to allowing concurrent editing using diff's:

  • Enables multiple developers to work on the same file(s)
  • Diff's allows the changes by each developer to be added and logged
  • Diff's only update the parts of the files which have changed
    • Ensures small data is transferred meaning:
      • Transfers are small ‐ thus, traffic on the network is reduced, an important consideration in businesses
      • Small transfers are fast ‐ ensuring there is little to no down time for developers working on the files
      • The small Fast transfers ensure synchronizations are made quickly, leading to all developers having quick access to the latest files
